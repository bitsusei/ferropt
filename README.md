# Ferropt

"Ferr-", a Latin prefix for iron, and "opt" from "optimization":
**Ferropt**, *Rust*-inspired embeded scripting language featuring reduced keywords
and flexible extensibility of modularity to various applications.
It also supports programming like predicates and processors.
In certain scenarios, its runtime should support parallelism and optimized performance.

Other major features include that it should be highly encapsulated,
that those scripts must not be compiled into machine language,
which could be executed directly by an operating system.
Instead, those should be interpreted binaries in an isolated environment
that each script even cannot know the existence of other script executing instances.
